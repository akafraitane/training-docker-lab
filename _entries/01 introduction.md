Welcome to the **Container Fundamentals Workshop**. This comprehensive hands-on workshop will guide you from zero to production-ready containerization skills using Podman and Docker.

In this workshop, you'll master:

- Understanding container fundamentals and architecture
- Installing and configuring Podman/Docker
- Building optimized container images
- Managing containers in production
- Networking and persistent storage
- Multi-container applications
- Security best practices and hardening
- Troubleshooting and debugging techniques

You can use this guide as a complete container tutorial and study material to build solid containerization skills from the ground up.

**Duration:** Approximately 480 minutes (8 hours)

**Target Audience:** Beginners with basic command-line familiarity

**Author:** Abdoul-Hakim Afraitane
