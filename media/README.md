# Media Directory

Place workshop images and media files here.

## Recommended structure:

```
media/
├── screenshots/
│   ├── podman-desktop.png
│   ├── container-architecture.png
│   └── network-diagram.png
├── diagrams/
│   ├── vm-vs-container.svg
│   └── multi-tier-app.svg
└── logos/
    ├── docker-logo.png
    └── podman-logo.png
```

## Usage in workshop

Reference images in markdown entries:

```markdown
![Container Architecture](./media/diagrams/container-architecture.png)
```
